Javascript Conditionals 101

Comparisons

Comparison and Logical operators are used to test for true or false

Comparison Operators

Comparison operators are used in logical statements to determine equality or difference between variables or values.

Given that x = 5, these are the comparison operators and what they do:

                
==   Equal to   x == 5 is true

                
===   Equal value and equal type   x === "5" is false

                
!=   Not equal   x != 6 is true

                
!==   Not equal value or not equal type   x !== "5" is true

                
>   Greater than   x > 2 is true

                
<   Less than   x < 5 is false

                
>=   Greater than or equal to   x >= 4 is true

                
<=   Less than or equal to   x <= 6 is true

Logical Operators

Logical operators are used to determine the logic between variables or values.

Given that x = 6 and y = 3, the table below explains the logical operators:

                
&&   and operator   (x < 10 && y > 1) is true

                
||   or operator   (x == 5 || y == 5) is false

                
!   not operator   !(x == y) is true
If Conditionals

Conditionals are used to do different actions depending of different decisions, this type of conditional need to have a parameter passed, if the parameter results true the action will be executed, but if it is false, nothing will happen.

The syntax is:

                
                    if (condition) {
                        //  block of code to be executed if the condition is true
                    }

Example:

                
                    let age = 10 ;
                    
                    if (age >= 10){
                        console.log("You are 10 years old or older");
                    }

                    // As the condition is true the code inside the if will be executed
Else Conditionals

The else conditional is used whith the if conditional to execute code when the condition inside the if is false. By using a if / else structure we can specify a block of code to execute when the if is true and another for when the if condition is false.

The syntax is:

                
                    if (condition) {
                        //  block of code to be executed if the condition is true
                    } else{
                        //  block of code to be executed if the condition is false
                    }

Example:

                
                    let age = 9;
                    
                    if (age >= 10){
                        console.log("You are 10 years old or older");
                    } else {
                        console.log("You are younger than 10 years old");
                    }

                    // As the age variable is 9, the condition is false so the 
                    // Code executed will be the one inside the else conditional
Else if Conditionals

The else if conditional is used to specify a new condition if the first one is false

The syntax is:

                
                    if (condition1) {
                        //  block of code to be executed if condition1 is true
                    } else if (condition2) {
                        //  block of code to be executed if the condition1 is false and condition2 is true
                    } else {
                        //  block of code to be executed if the condition1 is false and condition2 is false
                    }

Example:

                
                    let age = 9;
                    
                    if (age >= 10){
                        console.log("You are 10 years old or older");
                    } else if (age == 9) {
                        console.log("You are 9 years old")
                    } else {
                        console.log("You are younger than 10 years old");
                    }

                    // As the condition is true for the else if conditional,
                    // The code inside him will be executed
Switch Conditionals

The switch statement is used to perform different actions based on different conditions.

The syntax is:

                
                    switch(expression) {
                        case x:
                            // code block
                            break;
                        case y:
                            // code block
                            break;
                        default:
                            // code block
                    }

Code Explaination

  • The switch expression is evaluated once.
  • The value of the expression is compared with the values of each case.
  • If there is a match, the associated block of code is executed.
  • If there is no match, the default code block is executed.

Break Keyword

When JavaScript reaches a break keyword, it breaks out of the switch block.
This will stop the execution of inside the block.

It is not necessary to break the last case in a switch block. The block breaks (ends) there anyway.
If you omit the break statement, the next case will be executed even if the evaluation does not match the case.

Default Keyword

The default keyword specifies the code to run if there is no case match

Example:

                
                    let age = 9;
                    let text;

                    switch (age) {
                        case 9:
                            text = "You are 9 years old";
                            break;
                        case 10:
                            text = "You are 10 years old";
                            break;
                        default:
                            text = "You arent 9/10 years old";
                    }
                    
                    // The case's 9 code will be executed this time

Common Code Blocks

Sometimes you will want different switch cases to use the same code.
In this example case 4 and 5 share the same code block, and 0 and 6 share another code block

                
                    let age = 10;
                    let text;

                    switch (age) {
                        case 9:
                        case 10:
                            text = "You are 9/10 years old";
                            break;
                        case 7:
                        case 8:
                            text = "You are 7/8 years old";
                            break;
                        default:
                            text = "Enter a valid age";
                    }

                    // Code inside case 9 / 10 executed

Switching Details

  • If multiple cases matches a case value, the first case is selected.
  • If no matching cases are found, the program continues to the default label.
  • If no default label is found, the program continues to the statement(s) after the switch.

Strict Comparison

Switch cases use strict comparison (===).
The values must be of the same type to match.
A strict comparison can only be true if the operands are of the same type.

                
                    let age = "9";
                    let text;
                    
                    switch (x) {
                        case 9:
                            text = "You are 9 years old";
                            break;
                        case 10:
                            text = "You are 10 years old";
                            break;
                        default:
                            text = "Not a valid age";
                    }
                
                    // As the type of age is a string and the type of every case is a int,
                    // no one will match the age so the default code will be executed.
